# FUTURE_DS_01

## Business Sales Performance Dashboard

This project analyzes online retail sales data to identify revenue trends, top-selling products, and regional sales performance using Power BI.

## Tools Used
- Power BI
- Online Retail Dataset

## KPIs
- Total Revenue
- Total Orders
- Total Customers

## Key Insights
- The United Kingdom generated the highest revenue.
- Revenue trends increased over time.
- Top-performing products contributed a significant portion of total revenue.

## Recommendations
- Focus marketing on high-performing products.
- Expand strategies in strong-performing regions.
- Improve sales in lower-performing markets.
